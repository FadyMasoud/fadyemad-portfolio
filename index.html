About me

I'm a software engineer specializing in backend architecture and AI integration, with 3+ years building real-world, large-scale platforms used across Egypt, Saudi Arabia, Oman, and the USA.

My core stack is PHP 8.x / Laravel 9–10 and Node.js 16–20, with deep experience in RESTful API design, database engineering (MySQL & PostgreSQL), WebSockets, JWT/OAuth, and CI/CD pipelines.

More recently I've been working on full-stack AI integration — connecting GPT-4o, Gemini, and Claude through a unified OpenRouter pipeline, engineering intelligent prompts with guardrails, and exploring LLM vision analysis of architectural design images.

Beyond engineering, I teach Laravel, Python, PHP, and Node.js at NTI and ideaspace, mentoring CS graduates and students aged 13–19 on clean code, APIs, debugging, and testing fundamentals.

AI / LLM Engineering
Problem
Clients needed multiple AI models (GPT-4o, Gemini, Claude) in a single pipeline with structured LLM vision outputs — plus a secure payment layer to monetize AI features.
Solution
Built a unified multi-model pipeline via OpenRouter with React/Node.js. Integrated Stripe for subscriptions, one-time charges, webhook-verified transactions, and refund flows following idempotency best practices. Engineered guardrailed prompts for consistent structured outputs.

EdTech / Platform

A psychological studies institute needed a digital learning environment with multi-provider video delivery, student progress tracking, and automatic grade calculation — replacing a fully manual offline process.
Solution

Institutional Workflow Management — Al Dhaheri Group, KSA
Enterprise / ERP
Problem
Al Dhaheri Group in Saudi Arabia needed to digitize internal service requests across Career Development, Internal Affairs, and Expenses Management — handling 50+ distinct service types submitted by employees across multiple branches.
Solution
Built a configurable multi-step approval workflow engine. Employees submit any of 50+ request types; the system routes each to the responsible department manager at the correct branch. Result
Replaced manual paper/email-based request handling across all departments. Full request lifecycle tracked digitally. Management gained real-time visibility into pending requests per department, reducing bottlenecks across branches.
