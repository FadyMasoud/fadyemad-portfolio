About me

I'm a software engineer specializing in backend architecture and AI integration, with 3+ years building real-world, large-scale platforms used across Egypt, Saudi Arabia, Oman, and the USA.

My core stack is PHP 8.x / Laravel 9–10 and Node.js 16–20, with deep experience in RESTful API design, database engineering (MySQL & PostgreSQL), WebSockets, JWT/OAuth, and CI/CD pipelines.

More recently, I've been working on full-stack AI integration — connecting GPT-4o, Gemini, and other LLMs through a unified OpenRouter pipeline, engineering intelligent prompts with guardrails, and exploring vision-based LLM analysis of architectural design images.

Beyond engineering, I teach Laravel, Python, PHP, and Node.js at NTI and ideaspace, mentoring CS graduates and students aged 13–19 on clean code, APIs, debugging, and testing fundamentals.

Full-Stack AI Integration Platform
AI / LLM Engineering
Problem
Clients needed to leverage multiple AI models (GPT-4o, Gemini, Claude) simultaneously without building separate integrations for each — and required reliable, structured outputs from LLM vision analysis of architectural design images.
Solution
Built a unified multi-model pipeline via OpenRouter with a React/Node.js stack. Engineered intelligent prompts with guardrails to enforce reliable structured outputs. Integrated ScrapingBee for web data enrichment and Hugging Face for specialized model access.
Result
Single pipeline supporting GPT-4o, Gemini, Claude, and Hugging Face models. LLM vision successfully extracts structured insights from architectural images. Guardrails reduce hallucination and enforce consistent output schema.

E-commerce / Real-time
